### HTTP 请求地址

请求地址前缀:

BASE_URL = https://drone.godouav.com/portal/%n%

> %n% 为节点id

### 请求头

所有请求必须包含access_token=$access_token 头

### 应答消息

应答采用json结构体

```json
{
	// 状态码
    "code": 0,
    "msg": "内容，暂未填充",
	// 任意结构体或数组， 无数据一般为空
    "data": {}
}
```

### 状态码

| 错误码 | 描述 |
|--------|------|
|0| 无错误  |
|1| 参数错误 | 
|2| 内部错误  |
|3| 未连接  |
